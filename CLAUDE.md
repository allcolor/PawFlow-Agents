# Tool usage rule

You have an MCP tool: consult_claude(question, model?, max_tokens?).

Call consult_claude when:
- you are uncertain, or
- the task is high-stakes (security, architecture, money), or
- you want a second opinion before committing to a plan.

After calling:
- summarize Claude's answer in 3-7 bullet points,
- state whether you follow it and why.
